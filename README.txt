Project Name: E-Commerce Backend API
GitHub Repository: https://github.com/salmaalbadry/ecommerce-backend.git

Description:
This is an E-Commerce backend application built using Node.js, Express, and MongoDB. It follows the MVC (Model-View-Controller) architecture and provides a RESTful API for managing users,products,categories,cart,and orders.

Project Structure:
controllers/ : Handles the business logic and processes incoming requests
models/ : Defines the database schemas using Mongoose
routes/ : Defines the API endpoints
middleware/ : Custom middleware functions (e.g., error handling)
utils/ : Helper utility functions
db.js : Database connection setup
app.js: Express application setup and route mounting
server.js : Application entry point and server initialization

Available API Endpoints:
/api/users : User registration, login, and profile management
/api/categories : Product categories management
/api/products : Product listing and details
/api/cart : Shopping cart management
/api/orders : Order processing and tracking
